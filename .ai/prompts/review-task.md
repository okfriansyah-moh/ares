# Review Task

## Use

Review implementation of Task {{TASK_NUMBER}} from `docs/PLAN.md`.

## Inputs

- `AGENTS.md`
- `docs/architecture.md`
- `docs/PLAN.md`
- Task number: `{{TASK_NUMBER}}`
- Implementation diff and validation output

## Instructions

1. Read `AGENTS.md`.
2. Read `docs/architecture.md`.
3. Read `docs/PLAN.md`, focusing on Task {{TASK_NUMBER}}.
4. Review implementation of Task {{TASK_NUMBER}} only.

## Check

- Compliance with `AGENTS.md`, `docs/architecture.md`, and Task {{TASK_NUMBER}}.
- Tests cover required behavior and reported validation commands passed.
- Security requirements are preserved.
- Complexity stays within scope and avoids speculative abstractions.

## Output

Findings only, ordered by severity. If no findings, say `No findings.`
