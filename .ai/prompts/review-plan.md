# Review Plan

## Use
Review a plan for completeness, sequencing, and executability.

## Inputs
- Current plan
- Source specification, if available
- Execution status or known issues

## Instructions
- Apply `.ai/skills/plan-management/SKILL.md`.
- Report findings as complete, missing, sequencing issues, oversized tasks, and recommendations.
- Preserve existing task numbers when suggesting changes.

## Check
- Missing work is tied back to the source.
- Oversized tasks have a clear split point.
- Recommendations are specific enough to act on.
