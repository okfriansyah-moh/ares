# Create Plan

## Use
Create an implementation plan from a specification or scoped request.

## Inputs
- Source specification or request
- Affected layers and constraints
- Existing architecture or plan context

## Instructions
- Apply `.ai/skills/plan-management/SKILL.md`.
- Read the full source before decomposing work.
- Produce `docs/PLAN.md` or `docs/PLAN-<feature>.md`.
- Sequence tasks so prerequisites appear before dependents.
- Include validation steps and any knowledge the tasks will need.

## Check
- Tasks are small and executable.
- Dependencies and validation are explicit.
- The plan can be followed without guesswork.
